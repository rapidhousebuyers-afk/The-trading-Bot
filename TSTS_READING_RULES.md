# TSTS Reading Order & Lessons Learned

## Chart Reading Order (CRITICAL)

**Always read in this order:**
1. **Timeframes: RIGHT to LEFT** (highest TF → lowest TF)
   - Right panel = highest timeframe (e.g., 15m, 4H)
   - Middle panel = middle timeframe (e.g., 5m, 1H)
   - Left panel = lowest/exexecution timeframe (e.g., 30s, 15m)

2. **Panes within each panel: BOTTOM to TOP**
   - Sniper Oscillator (bottom) → BS Detector/Histogram (middle) → Candle/Bokk Channel (top)

## Sniper Oscillator — Line Meanings

- **Blue line** = Overall strength / trend anchor. If blue is UP, the underlying trend is intact. If blue turns DOWN, the trend is weakening.
- **Pink line** = Short-term momentum. Fast-moving, shows immediate directional pressure.
- **Purple line** = Confirms direction. Pink crossing Purple = momentum shift.

**KEY PATTERN — Bullish Divergence (LOGO Trade signal):**
- Blue UP + Pink falling = "Dip within trend" = buying opportunity
- The V in a LOGO trade IS this divergence: pink drops creating the dip, blue stays up confirming the trend never broke
- When pink turns back up → momentum rejoins strength → LOGO entry triggers

**KEY PATTERN — Trend Weakness (warning):**
- Blue DOWN + Pink falling = trend and momentum both weakening → bearish, no long entry
- Blue DOWN + Pink UP = short-term bounce within weakening trend → risky, avoid

## Trade Type Identification — Updated

### LOGO Trade (Clean V)
- Higher TF: Bokk opening green (bullish context)
- Middle TF: Bokk closing, BS dimming — this is the "V dip"
- Lower TF: Fading/mixed dir bars, sniper blue curling up while pink dips — this is the V-bottom
- **Entry trigger:** Lower TF flips green as pink momentum rejoins blue strength
- **Stop:** Below the V-bottom
- **Target:** Momentum loss (watch pink fall again)

### GGG RGG LONG (Safety Trade)
- Bottom-up build: Dir Bars solid green → BS bright green → Bokk opening green → Price above EMAs
- All TFs building simultaneously, not a V-shape
- Higher TFs in Reset (dimming red) while lower TFs already green = high-value forming setup

### RRR G RR SHORT (Safety Trade)
- Bottom-up: Dir Bars solid red → BS bright red → Bokk opening red → Price below EMAs
- Higher TFs in Reset (dimming green) while lower TFs already red = high-value forming setup

### No Trade
- Mixed dir bars on all TFs = NO TRADE
- Conflicting sniper signals (blue going different directions across TFs)
- All indicators in transition, nothing locked

## Confidence Calibration

- All layers locked across all TFs = 85-95%
- Execution TF confirmed + higher TFs in Reset = 70-90%
- LOGO forming with blue/pink divergence confirmed = 70-75%
- Only one TF showing signal, others mixed = 40-55%
- All indicators in transition = 15-25% (No Trade)
- Mixed dir bars = cap at 20-30%

## Common Mistakes to Avoid

1. **Don't read timeframes left to right** — it's RIGHT to LEFT (high to low)
2. **Don't read panes top to bottom** — it's BOTTOM to top (sniper first)
3. **Don't call a trade "forming" when ALL indicators are mixed** — that's a transition zone, not a setup
4. **Dimming red ≠ bearish** — it means bearish momentum is exhausting. Context matters: is it a Reset (higher TF) or a flip starting (lower TF)?
5. **Closing Bokk after a run = consolidation, not necessarily reversal** — check if higher TF Bokk is still open
6. **The sniper blue line is the single most important line** — if blue is up, the trend is your friend. If blue is down, don't fight it.
